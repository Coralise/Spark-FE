# Spark Landing Page

A modern, responsive landing page built with Next.js, featuring a beautiful hero section, feature cards, and a clean design.

üåê **Live Demo**: [https://spark-fe-six.vercel.app/](https://spark-fe-six.vercel.app/)

### Local Development

1. **Clone the repository**
   ```bash
   git clone <your-repository-url>
   cd spark-fe
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the landing page.

### Production Build

To create a production build:

```bash
npm run build
npm start
```
